# EvolvexFitness

EvolvexFitness is a full-stack fitness management platform developed using Spring Boot, React, and MongoDB.

## Features
- Workout tracking
- Diet management
- Supplement scheduling
- Secure authentication
- Responsive dashboard
- Real-time fitness management

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Spring Boot
- REST APIs

### Database
- MongoDB

## Installation

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm run dev
